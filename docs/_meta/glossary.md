# 용어사전
> 최종수정: 2026-04-05 | 상태: 초기화

## 사용법
- 모든 문서에서 도메인 용어를 쓸 때 이 사전의 정의를 따른다
- 새 용어가 등장하면 여기에 먼저 등록 후 문서에 사용한다
- 같은 개념에 다른 단어를 쓰지 않는다 (예: "회원"과 "사용자"가 같은 뜻이면 하나로 통일)
