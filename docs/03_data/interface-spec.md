| IF-ID | 대상 시스템 | 연동 유형 | 방향 | 데이터 포맷 | 호출 시점 | 관련 기능 |
|-------|-----------|----------|------|-----------|----------|----------|
| IF-{SYS}-001 | | API / DB / 파일 / MQ / 배치 | 송신 / 수신 / 양방향 | JSON / XML / CSV / 고정길이 | 실시간 / 배치 / 이벤트 | F-{DOMAIN}-{NNN} |

## 인터페이스 상세 (IF-ID당 1섹션)

### IF-{SYS}-001: {인터페이스명}

**기본 정보**
| 항목 | 내용 |
|------|------|
| 대상 시스템 | 시스템명, 버전, 담당부서/담당자 |
| 연동 유형 | API 호출 / DB 직접연결 / 파일전송 / 메시지큐 / 배치 |
| 방향 | 송신 / 수신 / 양방향 |
| 호출 시점 | 실시간 / 배치 (주기: 매일 02:00 등) / 이벤트 트리거 |
| 관련 기능 | F-{DOMAIN}-{NNN} |
| 관련 화면 | SCR-{NNN} (있으면) |

**데이터 정의**
| 필드명 | 타입 | 필수 | 설명 |
|--------|------|------|------|
| | | | |

**요청/응답 구조** (API인 경우)
```
요청: POST /external/api/v1/xxx
{
  "field": "value"
}

응답 (성공):
{
  "result": "OK"
}

응답 (실패):
{
  "error": "code", "message": "설명"
}
```
